## ¿Qué es SubScanLTM?

Herramienta todo en uno para descubrir **bug hosts**, escanear subdominios, detectar CDN (Cloudflare, Cloudfront, Akamai...), analizar puertos, registros DNS y más. Compatible con **Termux, Ubuntu, Debian, Kali Linux y cualquier sistema con Python 3**.

## 🛠️ Funciones

| # | Módulo | Descripción |
|---|--------|-------------|
| 1 | **HOST SCANNER** | Detecta bug hosts HTTP/HTTPS válidos |
| 2 | **SUBFINDER** | Enumera subdominios desde múltiples fuentes |
| 3 | **IP LOOKUP** | Reverse IP + geolocalización |
| 4 | **PORT SCANNER** | Escanea puertos abiertos (comunes o personalizados) |
| 5 | **DNS RECORDS** | Registros A, AAAA, MX, NS, TXT, CNAME, SOA |
| 6 | **HOST INFO** | CDN, servidor web, certificado SSL, headers de seguridad |
| 7 | **FULL SCAN** | Todo lo anterior en un solo análisis |
